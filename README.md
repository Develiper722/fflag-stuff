# how to get the fflags.json file urself 
first download grabflags.py and converttojson.py

second get a github private access token https://github.com/settings/personal-access-tokens

leave everything on default

third put the pat (personal access token) into the token thing

forth run the script with the version u want to start from [Roblox Client Tracker](https://github.com/MaximumADHD/Roblox-Client-Tracker/commits/roblox/FVariables.txt)

fifth run converttojson.py 

and now u have the fflags.json file!
